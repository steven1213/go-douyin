# go-douyin 后端项目

## 项目介绍

go-douyin 是一个基于 Gin 框架的后端项目，主要用于抖音功能的探索实现。

## 项目技术栈

- Gin
- Gorm
- Golang
- MySQL
- Redis
- Docker

## 项目计划

- [x] 扫码登录

## 项目运行

### 1. 克隆项目

```bash
git clone git@github.com:steven1213/go-douyin.git
```

### 2. 安装依赖

```bash
go mod tidy
``` 

### 3. 运行项目

```bash
go run main.go
```
